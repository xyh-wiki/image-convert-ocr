# Image Convert & OCR Platform

A modern online tool for image format conversion, OCR text extraction, image compression, cropping, and resizing.

## Features
- Convert between PNG, JPG, WEBP, TIFF, GIF, BMP, PSD
- OCR text extraction (English, Chinese supported)
- Image compression (custom quality)
- Image cropping (X/Y/Width/Height)
- Image resizing (Width/Height)
- Clean UI with light/dark gradient background
- Fully responsive design
- Google Ads & SEO Ready

## Usage
Upload an image and choose one of the available tools.

## API Endpoints
- POST `/api/image/convert`
- POST `/api/image/ocr`
- POST `/api/image/compress`
- POST `/api/image/crop`
- POST `/api/image/resize`
