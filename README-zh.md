# 图片处理平台（格式转换 / OCR / 压缩 / 裁剪 / 调整尺寸）

一个现代化在线图片工具平台，支持图片格式转换、OCR 识别、压缩、裁剪、调整尺寸等功能。

## 功能特性
- 支持 PNG、JPG、WEBP、TIFF、GIF、BMP、PSD 格式互转
- 支持 OCR 文字识别（中英等多语言）
- 支持自定义压缩比例
- 支持裁剪指定区域（X/Y/宽/高）
- 支持自定义宽度与高度调整
- 现代化渐变 UI，模块化布局
- 完善的 SEO 与 Google Ads 支持
- 响应式移动端适配

## 使用方法
上传图片，然后选择需要使用的工具即可开始处理。

## API 接口
- POST `/api/image/convert`
- POST `/api/image/ocr`
- POST `/api/image/compress`
- POST `/api/image/crop`
- POST `/api/image/resize`
